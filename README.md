# Career Reflection Toolkit

A password-protected web app for college students to reflect on their career goals, track daily progress, and plan their professional journey. Built as a college project at **NIU**.

> No frameworks. No backend. Runs entirely in the browser.

## Features

- Role-based login — Admin and Student with separate passwords
- Daily check-in — mood, intentions, end-of-day notes
- Weekly review — star rating + guided reflection questions
- Goal setting — add goals, set deadlines, mark complete
- Career planner — target role, skills, and timeline
- Progress tracker — mood charts and goal progress bars
- Admin dashboard — view all 10 students, filter by course
- Data saved in browser localStorage — no server needed
